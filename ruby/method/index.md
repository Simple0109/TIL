### index

```ruby
"astrochemistry".index("str") # => 1
"regexpindex".index(/e.*x/, 2) # => 3
"character".index(?c) # => 0

"foobarfoobar".index("bar", 6) # => 9
"foobarfoobar".index("bar", -6) # => 9
```
